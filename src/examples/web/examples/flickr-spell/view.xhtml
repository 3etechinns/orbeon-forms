Flickr Spell 1 Usage. Enter some text and we'll pull for each letter a random image from Flickr. Implementation. The UI an interaction with the server is implemented 100% in XForms. The XForms UI sends the text you type in an XML document to the server which responds with an XML document that contains link to the images. The server-side code is implemented in an XPL pipeline, calls a web service provided to Flickr and caches the response to avoid sending the same query over and over to the Flickr. Credits.
